# Contributing

Thanks for helping improve UIQA.

## Development Setup

```bash
pnpm install
pnpm test
pnpm build
```

## Rules

Rules should be small, deterministic, and individually tested. Each rule should explain why the finding matters and provide a practical suggestion.

When adding a rule:

- Add clear types where needed.
- Add focused unit tests.
- Add or update examples.
- Update documentation when the behavior is user-facing.

## Pull Requests

Keep pull requests focused. Include tests for behavior changes and make error messages clear enough for beginners.
