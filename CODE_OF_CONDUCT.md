# Code of Conduct

UIQA follows the Contributor Covenant Code of Conduct.

## Our Standards

We aim to maintain a practical, respectful, and inclusive project environment.

Examples of encouraged behavior:

- Use welcoming and direct language.
- Respect different experience levels.
- Give constructive feedback.
- Focus disagreement on ideas and implementation details.

Examples of unacceptable behavior:

- Harassment or discriminatory language.
- Personal attacks.
- Trolling or sustained disruption.
- Publishing private information without permission.

## Enforcement

Project maintainers are responsible for clarifying standards and may remove comments, commits, issues, or pull requests that do not align with this code.
